Коды ошибок

Формат ошибки​

Все ошибки возвращаются в едином формате:

{
  "error": {
    "code": "PROJECT_NOT_FOUND",
    "message": "Проект с указанным ID не найден"
  }
}

HTTP-статусы​

КодСтатусКогда возникает
200OKЗапрос выполнен успешно
201CreatedРесурс создан
400Bad RequestОшибка валидации данных
401UnauthorizedJWT отсутствует или истёк
403ForbiddenНет доступа к чужому ресурсу
404Not FoundРесурс не найден
409ConflictРесурс уже существует или лимит достигнут
422Unprocessable EntityAI не смог интерпретировать команду
429Too Many RequestsПревышен лимит AI-запросов
500Internal Server ErrorОшибка на стороне сервера
