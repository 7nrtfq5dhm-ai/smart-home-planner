# Smart Home Planner

Карточка продукта

Версия: **1.0.0** · Среда: **Production** · Обновлено: **2025-05-01**

**Smart Home Planner** — веб-платформа для самостоятельного проектирования жилых помещений.
Пользователи создают планы квартир и домов в интерактивном редакторе, расставляют мебель и инженерные системы, используют AI-помощника для генерации и редактирования планировки через текстовые команды.

## Основные функции

- **Редактор планировки** — интерактивный холст для рисования стен и комнат
- **Каталог объектов** — мебель, техника, инженерные коммуникации
- **AI-помощник** — генерация и изменение планировки через текстовые команды
- **Инженерные слои** — электропроводка, водоснабжение, канализация
- **Экспорт проекта** — скачать готовый план
