# Smart Home Planner

**Версия:** 1.0.0 · **Среда:** Production · **Обновлено:** 2025-05-01

**Smart Home Planner** — веб-платформа для самостоятельного проектирования жилых помещений. Пользователи создают планы квартир и домов в интерактивном редакторе, расставляют мебель и инженерные системы, используют AI-помощника для генерации и редактирования планировки через текстовые команды.

## Проблема

Большинство людей не обладают профессиональными знаниями в архитектуре, поэтому проектирование жилья становится сложным и непонятным. Существующие альтернативы либо слишком сложны (AutoCAD, ArchiCAD), либо дорого стоят (услуги дизайнеров), либо не включают инженерные системы и AI-помощь.

## Основные функции

- **Редактор планировки** — интерактивный холст для рисования стен и комнат
- **Каталог объектов** — мебель, техника, инженерные коммуникации
- **AI-помощник** — генерация и редактирование планировки через текстовые команды
- **Инженерные слои** — электропроводка, водоснабжение, канализация
- **Экспорт проекта** — скачать готовый план
