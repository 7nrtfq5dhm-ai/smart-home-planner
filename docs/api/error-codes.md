# Коды ошибок

## Формат ошибки

Все ошибки возвращаются в едином формате:

```json
{
  "error": {
    "code": "PROJECT_NOT_FOUND",
    "message": "Проект с указанным ID не найден"
  }
}
```

## HTTP-статусы

| Код | Статус | Когда возникает |
|-----|--------|----------------|
| `200` | OK | Запрос выполнен успешно |
| `201` | Created | Ресурс создан |
| `400` | Bad Request | Ошибка валидации данных |
| `401` | Unauthorized | JWT отсутствует или истёк |
| `403` | Forbidden | Нет доступа к чужому ресурсу |
| `404` | Not Found | Ресурс не найден |
| `409` | Conflict | Ресурс уже существует или лимит достигнут |
| `422` | Unprocessable Entity | AI не смог интерпретировать команду |
| `429` | Too Many Requests | Превышен лимит AI-запросов |
| `500` | Internal Server Error | Ошибка на стороне сервера |

## Коды ошибок приложения

| Код | HTTP | Описание |
|-----|------|---------|
| `VALIDATION_ERROR` | 400 | Ошибка валидации полей запроса |
| `INVALID_OTP` | 400 | Неверный или истёкший OTP-код |
| `UNAUTHORIZED` | 401 | Требуется авторизация |
| `FORBIDDEN` | 403 | Нет доступа к этому ресурсу |
| `PROJECT_NOT_FOUND` | 404 | Проект не найден |
| `USER_ALREADY_EXISTS` | 409 | Email или телефон уже зарегистрирован |
| `PROJECT_LIMIT_REACHED` | 409 | Достигнут лимит 3 проектов для Free |
| `AI_COMMAND_AMBIGUOUS` | 422 | AI не смог понять команду |
| `AI_QUOTA_EXCEEDED` | 429 | Исчерпан дневной лимит AI-запросов |
