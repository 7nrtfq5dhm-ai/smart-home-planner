# Технологии хранения данных

## Сущности, акторы и характер взаимодействия

| Сущность | Атрибуты | Акторы | Характер |
|---------|---------|--------|---------|
| Пользователь | ID, Email, Хеш пароля, Тип подписки | Пользователь, Система аутентификации | OLTP: высокая консистентность, безопасность |
| Метаданные проекта | ID, ID владельца, Название, Дата, Статус | Пользователь, Backend | OLTP: частое чтение списка |
| План помещения | Стены, Комнаты, Объекты (JSONB) | Пользователь (редактор), AI-сервис | OLTP: большой объём, гибкая структура |
| Задача AI | TaskID, ProjectID, Промпт, Статус | Backend, AI-сервис, Брокер | OLTP: высокая запись, короткий TTL |
| Библиотека объектов | ID, Категория, 2D/3D модель | Пользователь (каталог) | Справочник: редкие изменения |

## Выбор технологий

| Данные | Технология | Обоснование |
|--------|-----------|-------------|
| Пользователи и метаданные проектов | **PostgreSQL** | ACID, строгая целостность, внешние ключи |
| Данные чертежа (холст) | **PostgreSQL JSONB** | Гибкая структура, поддержка индексов |
| Статусы AI-задач | **Redis** | Молниеносный доступ, не нужно долгое хранение |
| Файлы моделей (2D/3D текстуры) | **S3 Object Storage** | БД не подходят для больших бинарных файлов |
| Аналитика событий | **Data Lake** | Петабайты данных, колоночное хранение для OLAP |

## Ключевой паттерн: разделение «горячих» и «холодных» данных

При загрузке дашборда система запрашивает только метаданные — название, дату, статус. Сам чертёж (несколько МБ) хранится отдельно в `project_canvas_data` и загружается только когда пользователь открывает редактор.

## Интерфейс экранов → API

| Экран | Действие пользователя | HTTP | Endpoint |
|-------|----------------------|------|---------|
| Дашборд `/` | Загрузка списка проектов | `GET` | `/api/v1/projects` |
| Генерация `/generate` | Создать через AI | `POST` | `/api/v1/projects/generate` |
| Редактор `/editor/{id}` | Загрузить план | `GET` | `/api/v1/projects/\{projectId\}` |
| Редактор `/editor/{id}` | Отправить AI-команду | `POST` | `/api/v1/projects/\{projectId\}/ai-commands` |
| Редактор `/editor/{id}` | Сохранить | `PUT` | `/api/v1/projects/\{projectId\}` |
| Редактор `/editor/{id}` | Инженерный слой | `GET` | `/api/v1/projects/\{projectId\}/layers?type=ELECTRICITY` |
