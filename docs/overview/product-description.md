# Описание продукта

## Проблема

Современные люди всё чаще сталкиваются с необходимостью спланировать ремонт, построить дом или изменить планировку квартиры. Большинство обычных пользователей не обладают профессиональными знаниями в архитектуре и строительстве, что делает процесс проектирования жилья сложным и непонятным.

**Ключевые боли целевой аудитории:**
- Сложно визуализировать будущий ремонт без специальных инструментов
- Люди не знают, как правильно распределить пространство
- Частые ошибки при планировании приводят к лишним затратам
- Нужен простой сервис без обращения к дорогим дизайнерам

## Текущие альтернативы и их недостатки

| Решение | Недостаток |
|---------|-----------|
| Бумажные схемы и наброски | Нет точности, трудно исправлять |
| AutoCAD / ArchiCAD | Слишком сложны, требуют обучения |
| Услуги дизайнеров | Дорого, не всегда доступно |
| Простые приложения | Нет инженерных систем и AI-помощи |

## Концепция продукта

Smart Home Planner — это цифровая платформа, которая объединяет:
- **Интерактивный редактор** планировок на холсте
- **Каталог объектов** (мебель, инженерные коммуникации)
- **AI-помощник** для генерации и редактирования через текстовые команды
- **Слои коммуникаций** (электропроводка, водоснабжение, канализация)
- **Экспорт** готового проекта

## Целевая аудитория

- Люди, начинающие планировать ремонт или строительство
- Семьи, покупающие новое жильё
- Владельцы частных домов и коттеджей
- Начинающие дизайнеры и студенты
- Пользователи, которым нужно быстро создать понятный проект для обсуждения
