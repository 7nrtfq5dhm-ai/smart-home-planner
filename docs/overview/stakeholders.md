# Стейкхолдеры

## Ключевые участники проекта

| Стейкхолдер | Роль | Влияние |
|------------|------|---------|
| Пользователи (клиенты) | Создают проекты домов и квартир | Формируют спрос, дают обратную связь |
| Команда разработчиков | Создание и поддержка системы | Принимают технические решения |
| Менеджеры-консультанты | Взаимодействие с пользователями | Повышают доверие к продукту |
| Партнёрские компании | Поставщики строительных материалов | Расширяют каталог, участвуют в монетизации |
| Инвесторы / заказчики | Финансирование развития | Определяют стратегические приоритеты |
| Строительные организации | Исполнение проектов | Используют проекты для реальных работ |

## Потребности и ожидания

**Пользователи** ожидают:
- Простой инструмент без профессиональных знаний
- Быструю визуализацию будущего ремонта
- Лёгкое внесение изменений
- Подсказки от AI-помощника
- Расчёт необходимых материалов
