# Бизнес-цели и ключевые метрики

## Цели (SMART)

### Цель 1 — Запуск MVP
- **Что:** Минимально рабочая версия с редактором и AI-помощником
- **Состав:** Авторизация, создание проекта, редактирование стен, ограниченные AI-команды
- **Срок:** До июня 2026 года

### Цель 2 — Рост пользовательской базы
- **Что:** Привлечь первых активных пользователей
- **Измерение:** ≥ 300 зарегистрированных пользователей
- **Срок:** В течение 2 месяцев после запуска

### Цель 3 — Вовлечённость
- **Что:** Увеличить количество проектов и использование AI
- **Измерение:** ≥ 1 проект и ≥ 5 действий редактирования за сессию
- **Срок:** Через 3 месяца после запуска

### Цель 4 — Монетизация через подписку
- **Что:** Конверсия в Premium-подписку
- **Измерение:** ≥ 5% конверсия
- **Срок:** Через 3 месяца после запуска

### Цель 5 — Скорость проектирования
- **Что:** Сократить время создания базовой планировки
- **Измерение:** 30–40 минут вместо нескольких дней

## KPI (Дерево метрик)

```
Завершённые проекты (главная метрика)
├── MAU (Monthly Active Users)
│   ├── Новые пользователи (регистрации, конверсия лендинга)
│   └── Возвращающиеся (Retention, среднее время в системе)
├── Среднее число проектов на пользователя
└── % завершённых проектов

Монетизация
├── Конверсия в Premium (target ≥ 5%)
├── ARPU
└── Платные консультации менеджера

Качество продукта
├── Среднее время создания проекта (target: 30–40 мин)
├── NPS / CSAT
├── Количество незавершённых проектов (UX-проблемы)
└── Время отклика системы
```
