# Платформизация

## Замена компонентов на внешние платформы

| Задача | Платформа | Обоснование |
|--------|-----------|-------------|
| Приём платежей (Premium) | **ЮKassa** | Не нужно соответствовать PCI-DSS; данные карты вводятся на защищённой стороне платформы |
| Email/SMS-уведомления | **Mindbox** | Уведомления об окончании AI-генерации без собственной инфраструктуры |
| Файловое хранилище | **S3 Object Storage** | Хранение изображений планов и текстур без собственных серверов |

### Как работает интеграция с ЮKassa

1. Виджет ЮKassa встроен на фронтенд
2. Пользователь вводит данные карты **на стороне ЮKassa** — не на нашем сайте
3. Наш бэкенд получает асинхронный **webhook** об успешной оплате
4. Бэкенд обновляет статус пользователя: `FREE` → `PREMIUM`

## B2B-платформизация

### Целевая аудитория (B2B-клиенты)

- **Застройщики недвижимости** (ПИК, Самолёт, А101)
- **Мебельные ритейлеры** (Hoff, Леруа Мерлен, IKEA)
- **Агентства ремонта и дизайна интерьеров**

### Стратегия

Редактор и AI-генератор упаковываются в **white-label виджет (SDK)**. Партнёр встраивает наш планировщик на свой сайт под своим брендом.

**Сценарий для застройщика:**
- Покупатель выбирает квартиру на сайте застройщика
- Открывает наш виджет прямо там
- Видит 3D-модель квартиры, двигает стены, расставляет мебель
- Просит AI «сделать дизайн под ключ»

### Бизнес-модель B2B2C

```
Smart Home Planner
    │
    │  продаём API / white-label виджет
    ▼
Бизнес-партнёр (Застройщик / Магазин)
    │
    │  встраивает в свой сайт
    ▼
Конечный пользователь (покупатель квартиры)
```

### Монетизация

| Модель | Описание | Пример |
|--------|---------|--------|
| SaaS-подписка | Фиксированная оплата за white-label виджет | 100 000 руб/мес |
| Pay-as-you-go | Оплата за каждый AI-вызов (GPU — дорогой ресурс) | 5 руб за генерацию |
