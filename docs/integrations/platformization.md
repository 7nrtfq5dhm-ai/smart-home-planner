# Платформизация

## Внешние платформы

Вместо разработки собственных решений используются готовые платформы:

| Задача | Платформа | Обоснование |
|--------|-----------|-------------|
| Приём платежей | **ЮKassa** | Не нужно соответствовать PCI-DSS; данные карты вводятся на защищённой стороне платформы |
| Email/SMS-уведомления | **Mindbox** | Уведомления об окончании AI-генерации без собственной инфраструктуры |

### Как работает оплата через ЮKassa

1. Виджет ЮKassa встраивается на фронтенд
2. Пользователь вводит данные карты на защищённой стороне ЮKassa — не на нашем сервере
3. Наш бэкенд получает асинхронный webhook об успешной оплате
4. Бэкенд обновляет статус подписки пользователя в БД: `Free` → `Premium`

## B2B-платформизация

### Целевые партнёры

- **Застройщики** (ПИК, Самолёт, А101) — встроить планировщик на страницу квартиры
- **Мебельные ритейлеры** (Hoff, Леруа Мерлен) — покупатель сразу расставляет мебель
- **Агентства ремонта** — показать клиенту результат до начала работ

### Продуктовая стратегия

Редактор и AI-генератор упаковываются в **белый лейбл (white-label)** — партнёр встраивает наш инструмент на свой сайт под своим брендом.

### Бизнес-модель B2B2C

```
Smart Home Planner
    │
    │ продаём API/виджет
    ▼
Бизнес-партнёр (Застройщик / Магазин)
    │
    │ встраивает в свой сайт
    ▼
Конечный пользователь (покупатель квартиры)
```

### Монетизация

| Модель | Описание | Пример |
|--------|---------|--------|
| SaaS-подписка | Фиксированная оплата за white-label виджет | 100 000 руб/мес |
| Pay-as-you-go | Оплата за каждый AI-вызов | 5 руб за генерацию |
