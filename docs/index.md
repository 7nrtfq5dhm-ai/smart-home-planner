# Smart Home Planner 🏠

> **Умный планировщик домов и квартир** — цифровая платформа для проектирования жилых помещений с AI-помощником

## О проекте

Smart Home Planner позволяет обычным пользователям самостоятельно создавать планы квартир и домов без профессиональных знаний в архитектуре. Система объединяет визуальный редактор, каталог мебели и инженерных систем, и AI-помощника в одном интерфейсе.

## Технологический стек

- **Frontend:** React, Canvas API
- **Backend:** REST API (OpenAPI 3.0)
- **Async:** RabbitMQ (AsyncAPI 2.6)
- **БД:** PostgreSQL, MongoDB, Redis, S3
- **AI:** AI-микросервис через очередь сообщений
